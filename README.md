# 🛒 React Shopping Cart App

A modern and responsive Shopping Cart application built using **React.js**. This project allows users to browse products, add items to the cart, update quantities, remove items, and view the total price in real time.

---

## 🚀 Features

* 🛍️ Display product list dynamically
* ➕ Add products to cart
* ➖ Increase or decrease product quantity
* ❌ Remove items from cart
* 💰 Calculate total price automatically
* 📱 Fully responsive UI
* ⚡ Fast and smooth user experience

---
